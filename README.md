# PING PONG #
## Author ##
 Peter Achacha
## Specifications
* A number divisible by three causes an output of 'Ping'. 
* A number divisible by five causes an ouput of 'Pong'.
* A number divisible by both three and five causes an output of 'Ping pong'.
